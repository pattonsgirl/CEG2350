**It's Complicated**

Monitors
- Resolution - [Wiki](https://en.wikipedia.org/wiki/Display_resolution#Computer_monitors)
- NVIDIA GSync vs AMD FreeSync
    - are adaptive sync technologies that eliminate screen tearing and reduce stutter by matching monitor refresh rates to GPU frame output
- HDR - High Dynamic Range
    - increases the range of luminosity in images, videos, and displays, allowing for brighter highlights, deeper shadows, and more vibrant colors than Standard Dynamic Range (SDR). It enhances contrast and detail, often used with Wide Color Gamut (WCG) to create more lifelike visuals in gaming, photography, and TV
    - Requires capable hardware (often OLED or Mini-LED) to display deeper blacks and brighter white highlights.

Machine Learning / AI
- NVIDIA CUDA Cores vs AMD Stream Processors
- Serve as the core component of a GPU for rendering graphics and handling compute tasks.

TPM - Trusted Platform Module
> is a specialized, tamper-resistant hardware chip on a computer's motherboard or processor that enhances system security by storing cryptographic keys, passwords, and digital certificates. It provides hardware-based authentication and ensures platform integrity, required for features like Windows 11 BitLocker encryption and Hello.

Secure Boot
> is a UEFI firmware security feature that ensures only trusted, digitally signed software loads during a computer's startup, protecting against malware and ransomware

Anti-Cheats
- [Wiki - How are people cheating in online games](https://en.wikipedia.org/wiki/Cheating_in_online_games)
- Common systems:
    - [Easy Anti-Cheat](https://www.easy.ac/en-US)
    - [BattlEye](https://www.battleye.com/)
    - [Anti-Cheat Expert](https://intl.anticheatexpert.com/)
    - [Riot Games AntiCheat](https://playvalorant.com/en-us/news/tags/anticheat/) (Valorant)
    - [EA Javelin](https://www.ea.com/news/introducing-ea-javelin-anticheat) Battlefield
- Key aspects:
    - Kernel-Level Access: Many modern anti-cheats operate at the kernel level (ring 0) of the operating system to detect advanced hacks, which can sometimes raise privacy concerns.
    - Functionality: They prevent cheating by scanning for malicious software, monitoring system memory, and verifying game file integrity.
    - Not always cross-platform compatible (making some games unplayable on systems that don't have support for the anti-cheat software required to play the game)
    - Machine learning is under investigation to look for "sus" behavior
